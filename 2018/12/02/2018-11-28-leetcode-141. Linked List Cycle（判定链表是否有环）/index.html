leetcode-141. Linked List Cycle（判定链表是否有环）

问题描述：

  • Given a linked list, determine if it has a cycle in it.

思路：

  • 采用快慢指针法，快指针每次前进两个节点，慢指针每次前进一个节点

  • 无环条件：node.next=NULL

  • 有环条件：快指针指向的节点=慢指针指向的节点
